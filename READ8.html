JavaScript Loops

Loops are handy, if you want to run the same code over and over again, each time with a different value. Instead of writing: text += cars[0] + "
"; text += cars[1] + "
"; text += cars[2] + "
"; text += cars[3] + "
"; text += cars[4] + "
"; text += cars[5] + "
";
You can write: var i; for (i = 0; i < cars.length; i++) { text +=cars[i] + "
" ; }

JavaScript supports different kinds of loops:

  1. for - loops through a block of code a number of times
  2. for/in - loops through the properties of an object
  3. for/of - loops through the values of an iterable object
  4. while - loops through a block of code while a specified condition is true
  5. while - loops through a block of code while a specified condition is true
